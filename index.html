About me

・Hi my name is George Ciobanu, born and raised in Romania, currently living in Japan Yokohama city. I'm a self taught 3D designer and front-end web developer.
・As an idealist I believe nothing is impossible, I always try to find solutions to any problem in a creative and meaningful way! I also believe using imagination with creativity, allows us to grow to unlimited heights.
・I started this wonderful journey in the design world when I was a high school student, using at that time AutoCad AutoDesk 2008 software, mainly doing 3D designs of various components using 2D blueprints.
・In 2019 I decided to challenge myself, this time using Blender software I started focusing again on 3D Modeling, approaching new styles like Voxel art, environmental components, mostly Hard Surface Modeling!
・But my interests in design didn't stop there, and I started studying Front-end development, web design using mainly HTML, CSS, JS.

goals

My goals are to become a full-fledged 3D designer, web designer, front-end developer, continuing to learn, grow and strive to become the best version of myself! And hopefully, one day becoming able to help others achieve their dreams of becoming future designers!
